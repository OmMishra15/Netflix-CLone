# 🎬 Netflix India Clone

A responsive front-end recreation of the Netflix India landing page built for learning semantic HTML5 layouts, modern CSS styling techniques, and advanced layout structures.

---

## ✨ Features

* **High-Fidelity UI Replication:** Precision layout styling that accurately mirrors the authentic Netflix India landing page theme, structural spacing, and distinctive typography scale.
* **Dynamic Media Layering:** Integrated absolute-positioned HTML5 `<video>` elements that sync smoothly and loop seamlessly behind device frames to replicate a real-world streaming viewport.
* **Interactive Frontend Elements:**
  * Clean, interactive Frequently Asked Questions (FAQ) components optimized with hovering transitions.
  * Stylized language selector options (`<select>`) implemented across the main navigation layout and the footer directory.
* **Advanced Layout Engineering:** Built using a structural combination of **CSS Flexbox** for component distribution and alignment, and **CSS Grid** for the multi-column footer tree (`grid-template-columns: 1fr 1fr 1fr 1fr`)

---

## 🛠️ Tech Stack & Concepts Practiced

* **Structure:** Semantic HTML5 (`<nav>`, `<section>`, `<footer>`, and descriptive class architecture)
* **Styling:** CSS3 (Custom Google Fonts, absolute positioning math, structural pseudo-selectors like `:first-child` and `:nth-child()`)
* **Media Handling:** Direct embedding of low-overhead loop-muted hardware assets (`.m4v` formats) natively overlapping visual vectors

## ⚙️ Local Setup and Installation

Follow these quick steps to clone the repository and run this project locally on your machine:

### 1. Clone the Repository

Open your terminal and run the following command to download the code:

```bash
git clone https://github.com/OmMishra15/Netflix-CLone.git
```

### 2. Navigate into the Project Directory

```bash
cd Netflix-CLone
```

### 4. Open and Run the Project

Because this is a pure frontend project built entirely on static web tools, you do not need to install heavy backend servers or build configurations. You can run it instantly:

* **Method A (VS Code Live Server):** Open the project workspace folder in Visual Studio Code, right-click `Index.html`, and select **Open with Live Server** to view changes live as you edit code.


* **Method B (Direct Local Launch):** Locate the project folder on your computer directory and double-click `Index.html` to run it automatically inside your preferred web browser (Chrome, Safari, Firefox, Edge).
